== Description ==

Non Profit is a fresh and airy theme that will introduce your content in a clear and well-organized manner. Not too fancy and overcrowded with visual elements, this theme will be perfect for those, who want their ideas to be presented in the spotlight. At the same time, you can give way to your creativity and change the look of your website in a few clicks, as this feature-rich theme comes packed with really simple and handy customization options. Developed in accordance with the latest theme design trends, Non Profit will be a professional solution for a whole range of various websites - from personal blogs to research centers, educational facilities, funds, science news, travel guides and much more. Live demo: http://wp-demo-theme.bestweblayout.com/non-profit/

== Installation ==

Manual installation:

1. Upload the `Non Profit` folder to `/wp-content/themes/` directory

Installation using "Add New Theme"

1. Go to your Dashboard and select Themes -> Add New in the menu.
2. Type "Non Profit" and click "Search".
3. Click the 'Install' button to open the theme's repository listing.
4. Click the 'Install' button.

Activation and Use

1. Activate the Theme using the 'Theme' menu in WordPress.

== Theme Notes ==

= Slider =

Once you install the theme, you will see an "Add to the slider" block in the Admin Panel => Posts => Add new (Edit post). This block will be located above the "Set featured image" metabox on the right. In this block, you will see a checkbox. To have this post added to the slider, please mark this checkbox and save the changes. 
Every post has its own metabox, so if you mark the checkbox of another post, the featured image of that post will be added to the slider as well, and the slider will have two slides. You can have as many posts as you want.
The slider is located on the main page. It consists of four parts: title (post title), image (post featured image), quote (the brief description of the post displayed in the slider), and the button ("Learn more" button that allows users to read the slider post completely).

= Post Thumbnail Functionality =

For post thumbnails, we recommend using images with the width equal to 560px.

= Post Formats: Aside, Image, Link, Quote =

Non Profit WordPress theme only supports these post formats. Please style them in a certain way. Other post formats will be displayed as regular posts.

= Widgets =

Non Profit includes a standard Widget. The following widgets are shown by default: recent posts, recent comments, archives, categories.

== Frequently Asked Questions == 

= How can I change language from English into Russian? =

Change the language in the admin settings screen. Settings > General > Site Language.

= How can I make use of custom-background? =

You can easily change background color through Admin UI (Dashboard): use the  Appearance menu > Customize > Colors > Background color.

= How do I set custom header image? =

You can set custom header image on your website from your Admin UI (Appearance). Please select Menu header => Choose Image => Browse, and then click 'Upload.' Please note that max size of the header image is 960px * 82px.

= How do I add a featured image? =

You can do this in Posts section > Add new or edit existing post > Set featured image.
